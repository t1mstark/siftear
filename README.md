# Siftear

Siftear is a mobile-first OSINT source platform for organizing public research links.

## Features
- Language picker (EN/DE/FR)
- Global search + category filters
- Curated OSINT source library
- Favorites + Research Board
- Local notes (browser storage)
- JSON export
- Static hosting ready (GitHub Pages)

## Dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
